What Am I Watching is a simple webserver to save ratings about stupid movies and series

This was done because I didn't found anything realy similar that let me save movies/series rating.

All web apps that provided a similar behavior were public and the idea here is to save ratings and comments between me and my gf.

Knowing this, I've built a simple go application that I then self host and every time we see a stupid movie/serie we add our ratings so we can then laugh about how bad some movies/series are
